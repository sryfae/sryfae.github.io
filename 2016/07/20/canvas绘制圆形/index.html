H5学习之canvas——绘制圆形

用canvas绘制圆形需要经过几道步骤：

  • 开始创建路径

    首先，使用图形上下文对象的beginPath方法，该方法的的定义如下所示：

    context.beginPath();

    该方法不使用参数。通过调用该方法，开始路径的创建。在几次循环地创建路径的过程中，每次开始都要调用beginPath函数。

  • 创建圆形路径

    创建圆形路径是，需要使用图形上下文对象的arc方法。该方法的定义如下所示。

    context.arc(x,y,radius,startAngle,endAngle,anticlockwise)

    该方法使用六个参数，x为绘制圆形的起点横坐标，y为绘制圆形的起点纵坐标，radius为圆形半径，startAngle为开始角度，endAngle为结束角度。anticlockwise为是否按顺时针方向进行绘制。在Canvas API中，绘制半径与弧时指定的参数为开始弧度与结束弧度，如果你喜欢使用
    角度，请使用如下所示的方法将角度转换为弧度。

    var radians = degree*math.PI/180

    其中Math.PI表示角度为180度，Math.PI*2表示角度为360度。

  • 关闭路径

    路径创建完成之后，使用图形上下文对象的closePath方法将路径关闭。该方法定义如下所示。

    context.closePath();

    将路径关闭后，路径的创建工作就完成了，但是请注意，这是只是路径创建完毕而已，还没有真正绘制任何图形。

  • 设定绘制样式，进行图形绘制。

    context.fillStyle = ‘rgba(255,0,0,0.25)’;

    context.fill();

    与绘制矩形方法一样。

function draw(id){
    var canvas = document.getElementById(id);
    if(canvas == null) return false;
    var context = canvas.getContext('2d');
    context.fillStyle = '#eef';
    context.fillRect(0,0,400,300);
    var n = 0;
    for(var i=0;i<10;i++){
        context.beginPath();//开始创建路径
        context.arc(i*25,i*25,i*10,0,Math.PI*2,true);
        context.closePath();
        context.fillStyle = "rgba(255, 0, 0, 0.35)";
        context.fill();
    }
}
